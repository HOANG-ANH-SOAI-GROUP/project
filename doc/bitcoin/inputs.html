WHAT ARE BITCOIN INPUT AND OUTPUT

In Bitcoin and its altcoin copies, every transaction requires at least one input and one output.

A Bitcoin transaction input is a Bitcoin address (aka sender) from which the money has to be transferred, and a transaction output is an address (aka recipient) to which the money has to be credited.

Each time you make a transaction, its input values must be collected from the sender(s), and all output values must be defined by you.

Source: https://www.thebalance.com/how-does-a-bitcoin-transaction-work-391213
